# Pstatus

Display the HTTP Status messages.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'pstatus'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pstatus

## Usage

    $ Pstatus.get_status_msg(200) # OK
    $ Pstatus.get_status_msg(201) # Created
